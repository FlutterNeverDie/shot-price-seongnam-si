# 💉 이 주사 ∙ 어디가 싸요? (Shot Price) 프로젝트 인수인계 가이드

이 문서는 **"이 주사 ∙ 어디가 싸요?"** 프로젝트의 핵심 컨셉, 기술 스택, 구현 상세를 관리하고 다른 AI 에이전트나 개발자에게 프로젝트를 원활하게 인계하기 위해 작성되었습니다.

---

## 1. 프로젝트 개요

**"이 주사 ∙ 어디가 싸요?"**는 **토스 앱 내 웹뷰(App-in-Toss)** 환경에 최적화된 **비급여 예방접종 가격 비교 플랫폼**입니다. 병원마다 부르는 게 값인 비급여 항목(가다실 9가, 독감, 대상포진 등)의 정보 비대칭을 해결하여, 사용자가 우리 동네 최저가 병원을 리스트로 직관적으로 확인하도록 돕습니다.

* **핵심 가치**: "똑같은 주사, 이제 우리 동네 최저가로 맞으세요."
* **타겟 사용자**: 유료 예방접종 계획이 있으며, 합리적인 의료 소비를 지향하는 모든 토스 앱 사용자.
* **플랫폼**: 모바일 웹 (Toss Pro Max V2 UI/UX 적용).

## 3. 핵심 기능 및 요구사항 (리스트 중심)

### A. 전역 상태 관리 (`store/usePriceStore.ts`)
* **백신 필터**: `selectedVaccine`(가다실, 독감, 대상포진 등)에 따른 데이터 필터링.
* **지역 설정**: `selectedRegion`(시/군/구) 기반의 데이터 필터링 및 로컬 스토리지 저장.
* **가격 데이터**: 심평원 API 응답값을 `hospitalName`, `price`, `updateDate` 등으로 정제하여 관리.

## 3-1. 주요 기능 및 요구사항

### 3.2. 지역 및 백신 항목 선택 (Search & Filter)
- **항목 탭**: 가다실 9가, 대상포진, 독감 등 주요 비급여 항목을 탭 형태로 제공하여 즉시 전환 가능.
- **지역 설정**: `@toss/use-overlay`를 활용한 바텀 시트 형식으로 시/도, 시/군/구 지역 선택.
- **매핑 로직**: 사용자 검색어(예: 가다실)를 심평원 API 전용 비급여 코드(`npayCd`)로 변환하여 쿼리.

### 3.3. 최저가 랭킹 리스트 (Ranking List)
- **정렬**: 별도의 지도 API 없이, 심평원 API 응답값(`curAmt`)을 기준으로 **낮은 가격순** 리스트업.
- **정보 노출**: 병원명, 접종가, 데이터 업데이트 날짜 노출.
- **절약 지표**: 해당 지역 평균가 대비 절약되는 금액을 강조 배지(Badge)로 노출하여 사용자 효능감 증대.

### 3.4. 보상형 광고 및 상세 정보 해금 (Ad-Reward)
- **정보 잠금**: 리스트 상태에서는 병원의 상세 주소 뒷부분과 전화번호(`telno`)를 블러(Blur) 처리하여 비노출.
- **광고 시청**: 사용자가 '상세 정보 보기' 또는 '전화하기' 클릭 시 `@apps-in-toss/web-framework` 기반 보상형 광고 노출.
- **리워드 지급**: 광고 시청 완료(`userEarnedReward`) 시에만 상세 정보를 해금하고 전화 걸기 기능 활성화.

## 5. 개발 워크플로우

1.  **데이터 정제**: 심평원 API의 비급여 항목 코드를 앱 내 예방접종 카테고리와 매핑.
2.  **광고 연동**: `showFullScreenAd` 호출 후 성공 콜백에서만 상세 정보 상태를 `unlocked`로 전환.
3.  **예외 처리**: `isSupported()`가 `false`인 경우 광고 없이 즉시 정보를 노출하는 폴백 로직 구현.
| **데이터 API** | 건강보험심사평가원 비급여 진료비 API | `getNonPaymentItemHospDtlList` 오퍼레이션 사용 |

## 5-1. AI 개발을 위한 아키텍처 규칙 (Rules)

1. **Feature 격리**: `features/search`, `features/price-list`, `features/reward-ad`로 로직 분리.
2. **Logic-Less UI**: 모든 비즈니스 로직(API 호출, 정렬, 광고 핸들링)은 커스텀 훅(`hooks/`)으로 추출.
3. **Type First**: 새로운 기능 구현 전 `types/`에 인터페이스를 먼저 정의하여 AI 문맥 파악 유도.
4. **Fallback 방어**: 광고 SDK 미지원 환경(`isSupported: false`)에 대한 즉시 해금 로직 필수 포함.
