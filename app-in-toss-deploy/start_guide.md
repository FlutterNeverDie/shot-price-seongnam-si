### 💡 초기 세팅 가이드 (Quick Start)

가장 빠르게 위 환경을 구성하시려면 터미널을 열고 다음 순서대로 진행하세요.

1. **Vite + React + TS 기초 뼈대 생성**
   ```bash
   npx create-vite@latest ./ --template react-ts
   ```
2. **필수 라이브러리 및 토스 SDK 설치**
   ```bash
   npm install @apps-in-toss/web-framework @toss/use-overlay zustand framer-motion lucide-react
   ```
3. **개발 서버 실행**
   ```bash
   npm run dev
   ```
