# 콘솔 가이드

![](/assets/ads_flow_0.LhT_Rt7A.png)

## 1. 사업자 정보 등록하기

인앱광고를 연동하려면 먼저 사업자 정보를 등록해야 해요.\
사업자 정보가 등록되어 있어야 이후 단계인 약관 동의와 정산 정보 입력을 진행할 수 있어요.\
사업자 정보 등록 방법은 [다음 가이드](/prepare/register-business.html)를 참고해 주세요.

![](/assets/ads_console_1.ZvVyK31v.png)

## 2. 정산 정보 입력하기

인앱광고를 연동하면 광고 수익이 발생해요. 수익을 정산받으려면 정산 정보를 등록해야 해요.\
워크스페이스의 ‘정보’ 탭에서 정산 정보를 입력한 뒤 검토 요청을 해 주세요.\
검토에는 영업일 기준 평균 2~3일이 걸려요.

![](/assets/ads_console_2.DOKRp3_c.png)

![](/assets/ads_console_3.DKxcgAyw.png)

::: tip 꼭 확인해 주세요
예금주명은 통장 사본에 적힌 이름과 한 글자도 다르지 않게 입력해 주세요.\
다르게 입력하면 정산이 지연될 수 있어요.
:::

## 3. 광고 그룹 생성하기

앱인토스 인앱광고는 구글 애드몹 광고 정책을 따라요.\
정책을 지키지 않으면 광고가 제한되거나 중단될 수 있어요.

![](/assets/ads_console_4.C8VkpCW0.png)

### 광고 그룹 이름

광고 유형과 노출 위치를 함께 적으면 관리하기 쉬워요.

예시

* 메인\_전면광고
* 게임종료\_리워드광고

운영 중 여러 광고 그룹을 관리해야 하므로, 나중에 봐도 바로 이해할 수 있게 작성해 주세요.

### 광고 유형

**전면 광고**

* 화면 전체에 노출되는 광고예요.
* 별도의 리워드 설정은 필요하지 않아요.

**리워드 광고**

* 사용자가 광고를 시청하면 보상을 제공하는 광고예요.
* 예를 들어 아이템 지급, 게임 이어하기 같은 보상을 설정할 수 있어요.

**배너 광고**

* 미니앱 상단 또는 하단에 노출되는 광고예요.
* 배너 광고는 고정 형태로 제공 되기 때문에 개발 시 노출 되는 위치 등을 지켜주세요.

### 보상 설정

리워드 광고를 선택한 경우에만 입력해요.\
사용자가 실제로 받는 보상 이름과 수량을 정확히 입력해 주세요.

예시

* 서비스 내 보상 단위: 기회
* 수량 및 금액: 1

사용자 화면에 표시될 내용이므로, 서비스에서 사용하는 용어와 동일하게 작성하는 것이 좋아요.

### 미디에이션 설정

카테고리와 하위 카테고리는 ‘앱 정보’에 등록한 값으로 자동 설정돼요.\
다른 카테고리로 변경할 수 있지만, 이 변경이 앱 정보에 반영되지는 않아요. 광고 그룹 생성 시에만 적용돼요.

카테고리에 맞는 광고 네트워크가 자동으로 설정돼요.\
광고 네트워크와 폭포식 구조는 직접 수정할 수 없고, 생성 이후 설정된 값을 확인만 할 수 있어요.

::: tip 꼭 확인해 주세요

* 광고 그룹 ID는 구글에 등록되기까지 최대 2시간이 걸릴 수 있어요.
* 광고 그룹을 생성한 뒤 제공되는 [광고 그룹 ID로 개발](https://developers-apps-in-toss.toss.im/ads/develop.html)해 주세요.
  :::

## 4. 광고 성과 및 정산 내역

개발과 출시를 완료한 뒤에는 인앱광고 성과와 정산 내역을 확인할 수 있어요.

### **광고 성과**

기간과 운영체제(OS)를 선택해 다음 항목을 확인할 수 있어요.

* 총 광고 노출 수, eCPM, 총 예상 수익
* 성과 데이터는 성과 발생 익일 오전 4시에 업데이트돼요.

::: tip eCPM 이란?

eCPM은 광고가 1,000회 노출될 때 발생하는 예상 수익이에요.

예를 들어 총 노출 수가 300회이고, eCPM이 60,000원이라면

1,000회 노출 시 60,000원의 수익이 발생하므로, 1회 노출당 수익은 60원이에요.

300회가 실제로 노출되었으므로 예상 수익은 18,000원이에요.

:::

![](/assets/ads_console_5.BPWq4l_K.png)

### **정산 내역**

* 매월 1일부터 말일까지의 수익은 다음 달 1일에 업데이트돼요.
* 다음 달 1일에 확정된 수익은 해당 월 말일에 입금돼요.

정산 구조에 대한 자세한 내용은 [다음 문서](https://developers-apps-in-toss.toss.im/settlement/intro.html#_1-%E1%84%8B%E1%85%B5%E1%86%AB%E1%84%8B%E1%85%A2%E1%86%B8-%E1%84%80%E1%85%AA%E1%86%BC%E1%84%80%E1%85%A9-%E1%84%8C%E1%85%A5%E1%86%BC%E1%84%89%E1%85%A1%E1%86%AB-%E1%84%80%E1%85%AE%E1%84%8C%E1%85%A9)를 참고해 주세요.
