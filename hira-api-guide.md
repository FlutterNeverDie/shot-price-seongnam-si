# 🏥 건강보험심사평가원 비급여진료비 Open API 가이드

이 문서는 **"이 주사 ∙ 어디가 싸요? (Shot Price)"** 프로젝트의 핵심 데이터 소스인 [건강보험심사평가원_비급여진료비정보조회서비스](https://www.data.go.kr/data/15001700/openapi.do)를 분석하여 정리한 가이드입니다.

---

## 1. 서비스 개요
*   **서비스명**: 건강보험심사평가원_비급여진료비정보조회서비스
*   **목적**: 병원별로 자율성 있게 책정되는 비급여 항목(예방접종, 검사비 등)의 가격 정보를 투명하게 공개하여 국민의 의료 선택권을 보장함.
*   **데이터 범위**: 병원급 이상 의료기관(필수), 참여 중인 일부 의원급 기관.

## 2. 주요 기능 (Operations)

| 기능명 | 설명 | 활용 포인트 |
| :--- | :--- | :--- |
| **getNonPaymentItemHospDtlList** | **비급여 항목 병원 목록 조회** | **프로젝트 핵심.** 특정 지역의 특정 주사 가격 리스트를 가져옴. |
| **getNonPaymentItemCodeList** | 비급여 항목 코드 정보 조회 | 가다실, 홍역 등 검색어에 맞는 API 내부 코드를 찾을 때 사용. |
| **getNonPaymentItemRegionalSststList** | 비급여 진료비용 지역별 정보 | "내 주변 평균가보다 OO원 싸요" 메시지 생성 시 기준값으로 활용. |
| **getNonPaymentItemInscSststList** | 비급여 진료비용 종별 정보 | 대학병원 vs 일반의원의 가격 차이 통계 제공 시 활용. |

---

## 3. 핵심 검색 파라미터
`getNonPaymentItemHospDtlList` 호출 시 지역과 항목을 조합하여 쿼리할 수 있습니다.

*   **`sidoCd` (시도코드)**: 서울(110000), 경기(310000) 등 대분류 지역.
*   **`sggCd` (시군구코드)**: 강남구(110001), 용인시 수지구(311901) 등 상세 지역.
*   **`npayCd` (비급여항목코드)**: 특정 주사 코드. (예: `L001...`)
*   **`yadmNm` (병원명)**: 특정 병원 이름을 직접 검색할 때 사용.

---

## 4. 응답 데이터 (Response) 예시
API가 반환하는 주요 필드는 다음과 같습니다.

```json
{
  "item": {
    "yadmNm": "건강검진병원",         // 병원 이름
    "addr": "서울특별시 강남구 ...",    // 병원 주소
    "telno": "02-123-4567",          // 병원 전화번호 (광고 락 걸 대상)
    "npayKorNm": "가다실9프리필드",     // 비급여 항목명
    "curAmt": 210000,                 // 현재 책정 가격 (핵심)
    "minAmt": 190000,                 // 최저가 범위
    "maxAmt": 240000,                 // 최고가 범위
    "lastUpdtDt": "20240311"          // 데이터 업데이트 일자
  }
}
```

## 5. Shot Price 프로젝트 활용 전략

### A. 지역 및 백신 필터링
*   사용자가 상단 탭에서 **[가다실9가]**를 선택하고 지역을 **[서울 강동구]**로 설정하면?
*   시스템은 `npayCd`와 `sidoCd/sggCd`를 조합하여 최저가 순으로 정렬된 리스트를 보여줍니다.

### B. 가격 혜택 강조 (Benefit)
*   `getNonPaymentItemRegionalSststList`를 통해 해당 지역의 **평균가**를 미리 캐싱해둡니다.
*   "이 병원은 평균보다 **4만원 더 저렴해요!**"와 같은 매력적인 UI를 구성합니다.

### C. 수익화 모델 (Ads)
*   `curAmt`와 `yadmNm`은 리스트에서 즉시 공개합니다.
*   **전화 연결(`telno`)** 및 **상세 위치(`addr`)**는 보상형 광고 시청 후 `unlocked` 처리하여 사용자 액션을 유도합니다.
